위 표를 토대로 각 Segment에 대한 마케팅 비용을 계산할 수 있는데, 각 Segment에 대한 마케팅 비용은 변수의 기준 마케팅 비용의 합으로 정의한다.
예를 들어, 한 Segmentation이 (PhD, Married, Income = 20,001 ~ 30,000)으로 구성될 경우, 총 마케팅 비용은 50 + 50 + 300 = 400이 해당 Segment의 마케팅 비용이 된다.
